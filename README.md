# Обрезка ссылок и подсчет переходов с помощью Вконтакте

Удивительно просто преобразовать любую ссылку в короткую с доменом vk.cc, и подсчитать для нее количество переходов.
Для этого нужно запустить файл main.py c параметром в виде ссылки для преобразования,
для подсчета переходов запустить main.py с параметором в виде короткой ссылки с доменом vk.cc.

```bash
C:\python_scripts\API_03>python main.py https://dev.vk.com/ru/mini-apps/management/creating-new-apps?ref=old_admin_panel

Короткая ссылка:  https://vk.cc/cTNUK9

C:\python_scripts\API_03>python main.py https://vk.cc/cTNUK9

Количество откликов:  11
```

### Как установить
Создать мини-приложение вконтакте https://vk.cc/cTNUK9,
скопировать севрисный ключ и вставить в файл .env, присвоив переменной окружения VK_SERVICE_KEY, как в примере ниже

`VK_SERVICE_KEY=jf85heik58fydjdkfifundk7`

Python3 должен быть уже установлен. Затем используйте pip (или pip3, есть конфликт с Python2) для установки зависимостей:

```bash
pip install -r requirements.txt
```
